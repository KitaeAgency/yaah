YAAH! Demo

Welcome ! YAAH is a simple lib for handling various cases of AJAX requests. It uses data-attributes to do that. Here's some examples to show you how to use it.


Simple link

It only needs a class to work. The default behavior is to replace the clicked element by the data recieved.

Example

<a href="./path/to.html" class="yaah-js">Click Me!</a>
Click Me!

Button with data-ya-href

With the data-ya-href it can be used on almost any html element

Example

<button class="yaah-js" data-ya-href="./path/to.html">Click Me!</button>

Passing serialized data with data-ya-post

Example

<a href="./path/to.html" class="yaah-js" data-ya-post='{"test":"example"}' >Click Me!</a>
Click Me!

Handling location

You can choose precisely where to insert the data with data-ya-target to choose a target other than the clicked element himself.data-ya-location is used to define if you want it before/after/inside/... the chosen target

Defining the target with data-ya-target="#target"

Example

<a href="./path/to.html" class="yaah-js" data-ya-target="#after" >Click Me!</a>
<div id="#target" >Hey I'm a target!</div>
Click Me!
Hey I'm a target!

Inserting data after the target data-ya-location="after"

Select event type with data-ya-trigger

By default data-ya-trigger="always"

Example

<a href="./path/to.html" class="yaah-js" data-ya-trigger="always" >Click Me!</a>
Click Me!

Only on time with data-ya-trigger="once"

Example

<a href="./path/to.html" class="yaah-js" data-ya-trigger="once" >Click Me!</a>
Click Me!

Automaticaly with data-ya-trigger="autoload"

Example

<a href="./path/to.html" class="yaah-js" data-ya-trigger="autoload" >Click Me!</a>
Click Me!

Submit a form with data-ya-trigger="submit"

Example

<form class="yaah-js" data-ya-trigger="submit" action="./path/to.html">...</form>

Redirect after success with data-ya-redirect="path/to/url"

Example

<a href="./path/to.html" class="yaah-js" data-ya-trigger="autoload" >Click Me!</a>
Click Me!

Confirm before send data-ya-confirm="Message to show"

Example

<a href="./path/to.html" class="yaah-js" data-ya-confirm="JavaScript! Do you speak it?" >Click Me!</a>
Click Me!

Update url with data-ya-pushstate="/path/to/show"

Example

<a href="./path/to.html" class="yaah-js" data-ya-pushstate="/path/to/show" >Click Me!</a>
Click Me!

Loop requests with data-ya-timer="30" (in seconds and data-ya-trigger="autoload" only for now)

Example

<a href="./path/to.html" class="yaah-js" data-ya-timer="30" data-ya-trigger="autoload" >Click Me!</a>
Click Me!
